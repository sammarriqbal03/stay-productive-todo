# Stay Productive — To-Do List App

## 📌 Description
This is a responsive To-Do List web application designed to help users stay organized and productive. It supports multiple color themes, task management with local storage, and a live date & time display — all wrapped in a clean, modern UI.

## 🎯 Features

- Add Task
- Complete / Uncheck Task
- Delete Task
- 3 Beautiful Themes (Ocean, Sunset, Forest)
- Theme Persistence (saved in localStorage)
- Live Date & Time Display
- Fully Responsive Design
- Smooth Animations & Transitions
- GitHub Profile Link

## 🧱 Concepts Used

- HTML5 Semantic Structure
- CSS3 Flexbox & Grid Layout
- CSS Animations & Transitions
- Media Queries for Responsiveness
- JavaScript DOM Manipulation
- localStorage for Data Persistence
- Event Listeners for Interactivity

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Fonts (Poppins)
- Font Awesome Icons
